# Baby Learning Emoji Game

## Overview
The Baby Learning Emoji Game is an engaging mobile application designed for Android users, built using Kotlin. This game aims to teach young children about animals and objects through the delightful use of emojis, sounds, and supportive Bengali language features.

## Features
- 🐶 **Emoji Based Learning**: Children can learn about different animals and objects by interacting with vibrant emojis
- 🔊 **Sound Support**: Each emoji has an associated sound for auditory learning
- 🇧🇩 **Bengali Language Support**: Complete Bengali language support for young learners
- 👆 **Interactive Gameplay**: Tap emojis to hear animal names and sounds
