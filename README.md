## Baby Learning Emoji Game 🎮

একটি শিশু-বান্ধব শিক্ষামূলক অ্যাপ যেখানে শিশুরা বিভিন্ন প্রাণী এবং বস্তু শিখতে পারে।

### বৈশিষ্ট্য ✨
- 🎨 রঙিন ইমোজি
- 🔊 অডিও প্রফেশনাল
- 🇧🇩 বাংলা সাপোর্ট
- 🎮 ইন্টারঅ্যাক্টিভ গেমপ্লে
- 📱 মোবাইল অপটিমাইজড

### ডাউনলোড APK 📲

**সর্বশেষ APK ডাউনলোড করুন:**
👉 [GitHub Actions - Artifacts](https://github.com/ashik3e/baby-learning-app/actions)

1. সবচেয়ে সাম্প্রতিক রান খুঁজুন
2. "Artifacts" সেকশনে "BabyLearningApp-APK" ডাউনলোড করুন
3. APK ফাইল ডাউনলোড করুন এবং ডিভাইসে ইনস্টল করুন

### ইনস্টলেশন 📲

```bash
adb install BabyLearningApp-APK/app-debug.apk
```
